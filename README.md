# BPSR Auto-Accept Party Tool

Alat bantu otomatisasi ringan untuk menerima permintaan party di **Blue Protocol: Star Resonance** secara otomatis menggunakan deteksi gambar.

## ✨ Fitur Utama
- **Dark Mode UI**: Antarmuka modern yang nyaman di mata.
- **Focus Detection**: Hanya aktif jika jendela game sedang di depan (mencegah salah input).
- **Mini Mode**: Ukuran jendela bisa dikecilkan agar tidak menutupi layar game.
- **Log Real-time**: Memantau aktivitas deteksi secara langsung.

## 🛠️ Persiapan & Instalasi
1. Pastikan kamu memiliki [Python 3.10+](https://www.python.org/downloads/).
2. Instal pustaka yang diperlukan:
   ```bash
   pip install pyautogui pydirectinput keyboard pywin32
